# AS91906 Assessment Repository
Use complex programming techniques to develop a computer program.

## Assessment Conditions
- This is an individual assessment.
- You may use your lesson notes and previous practice work.
- You may NOT copy code from the internet or AI tools.
- All commits must be your own work.

## Project Options
Choose ONE:
1. Task Manager (text-based or GUI)
2. Interactive Game (text-based or GUI)

## Required Complex Techniques (minimum 2)
- User-defined functions
- Classes and objects
- Persistent data storage (JSON/file handling)
- Robust input validation
- Structured program design

## Evidence Required
- Completed program
- Testing table (expected, boundary, invalid)
- Short explanation of complex techniques used
